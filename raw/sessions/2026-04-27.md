**Lessons Learned:**
- Supabase edge function logs have eventual consistency — don't trust absence of logs in first pull; re-query before concluding a function wasn't invoked
- Truthy-object trap: `{success: false, error: "..."}` is truthy in JS — always check `.success` explicitly in routing/forwarding logic
- Telegram bot-to-user messaging requires the user to `/start` the bot first (403 otherwise) — this is easy to miss for agents added to group chats who never DM the bot
