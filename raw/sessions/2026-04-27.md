### Session (01:50)

**Context:** Performance fix for video loading on Qualia Solutions /work page + project reorder for featured section.

**Key Exchanges:**
- User reported videos load with black screen and lag when scrolling to projects — wanted immediate poster visibility
- User specified AI Boss Brainz, Sophia AI, Alexis Armenius Voice Agent, Axidex should be top featured projects

**Decisions Made:**
- Removed `opacity: ready ? 1 : 0` pattern from DeviceMockup3D.tsx — it was hiding the poster behind a transparent video element, causing the black screen. Poster now shows instantly.
- Changed video `preload` from `"none"` to `"metadata"` and bumped IntersectionObserver `rootMargin` from 200px → 1200px so videos prefetch ~1.5 viewports ahead of scroll position.
- Removed `comingSoon: true` from Sophia Zyprus and Alexis Voice Agent — both past their March 2026 launch date, so they now appear in the main featured rail.
- Reordered `projectOrder` in `data/projects.ts`: AI Boss Brainz → Sophia Zyprus → Alexis Voice Agent → Axidex → Peta as top 5.

**Lessons Learned:**
- The `opacity: 0` until `canplay` pattern is a common anti-pattern for video elements with poster images — it hides both the video AND the poster, causing a blank/black screen. Better to show the poster immediately and let the video paint over it naturally.
- Git rebase during deploy can silently revert fixes if upstream redesigned the same files — always verify changes survived after rebase before shipping.
