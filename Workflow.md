## Project Lifecycle

```
Lead/Demo -> Proposal -> Contract -> Design -> Build -> Test -> Deploy -> Handoff -> Retainer/Maintenance
```

### 1. Lead / Demo
- Initial meeting or demo with prospect
- Current pipeline: [[Jaber]], [[Aris]], [[Kounouz]], [[Kronospan]], [[Pita]], [[Innrvo]]

### 2. Proposal
- [[Sally]] or [[Fawzi Goussous]] drafts proposal
- Email exchange, scope definition

### 3. Contract
- Terms, payment schedule, milestones agreed
- Invoice structure defined (see [[Finances Overview]])

### 4. Design
- UI/UX, branding alignment
- Brand standards: distinctive fonts, cohesive palettes, fluid layouts

### 5. Build
- Stack: Next.js, React, TypeScript, Supabase
- Mobile: Expo (React Native) where applicable
- AI: VAPI, ElevenLabs, Retell AI, OpenRouter

### 6. Test
- Current in testing: [[Sofia]]
- QA, client review cycles

### 7. Deploy
- Vercel for web projects
- Expo builds for mobile
- Post-deploy checklist: HTTP 200, auth flow, console errors, API latency, UptimeRobot

### 8. Handoff
- Client training, documentation
- Access transfer

### 9. Retainer / Maintenance
- Monthly retainers for ongoing support
- Active retainers: [[Zyprus - Charalambous]], [[Armenius]], [[Giulio - Underdogsales]]
