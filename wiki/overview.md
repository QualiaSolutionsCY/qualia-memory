# Qualia Brain — Overview

> The second brain for Qualia Solutions, powered by the LLM Wiki pattern.

## What This Is

A persistent, compounding knowledge base where every article, meeting, transcript, and conversation makes the system smarter. Built on Obsidian + Claude Code.

## Architecture

- **Obsidian** = the IDE (visual interface, graph view, backlinks)
- **Claude Code** = the programmer (reads, writes, organizes knowledge)
- **Wiki** = the codebase (structured, interlinked markdown)
- **Qualia ERP** = source of truth for operations

## How Knowledge Flows

1. **Ingest** — Raw sources dropped into `raw/`, Claude processes into wiki pages
2. **Query** — Questions answered by traversing the wiki graph
3. **Capture** — Every Claude Code session auto-logged to `daily_log.md`
4. **Promote** — Daily extraction of key decisions/lessons into `memory.md`
5. **Lint** — Periodic health checks for orphans, contradictions, gaps
