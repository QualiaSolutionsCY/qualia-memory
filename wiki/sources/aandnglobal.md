# aandnglobal

> A&N Global — client website with about, contact, loyalty card pages built on Next.js 16.

## Overview

A Next.js 16 website for A&N Global. The site includes about, contact, and loyalty card sections with SEO support (robots.txt, sitemap). Built with Font Awesome icons, SWR for data fetching, and a custom error/404 handling setup.

## Tech Stack

- Next.js 16.1, React 19, TypeScript
- Tailwind CSS
- Font Awesome (icons)
- SWR (data fetching)
- Vercel (hosting)

## Status

- **Language:** TypeScript
- **Last updated:** 2026-04-11
- **Visibility:** Private

## Key Features

- About page
- Contact page
- Loyalty card system
- SEO setup (robots.txt, sitemap generation)
- Custom error and 404 pages
