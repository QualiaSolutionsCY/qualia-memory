# blackgoldunited

> Black Gold United Custom ERP system with full CI/CD pipeline

## Overview

A custom ERP system for Black Gold United built with Next.js and TypeScript. Features a comprehensive GitHub Actions CI/CD pipeline with validation, security auditing (CodeQL, npm audit), multi-node build testing, Playwright E2E tests, and automated staging/production deployment to Vercel. Includes custom Claude Code integration scripts for project analysis and context integrity checking.

## Tech Stack

- Next.js (with Turbopack)
- TypeScript
- Node.js 18/20 (multi-version CI)
- GitHub Actions (CI/CD)
- Vercel (deployment)
- CodeQL (security analysis)
- Playwright (E2E testing)
- ESLint
- Python 3.11 (automation scripts)
