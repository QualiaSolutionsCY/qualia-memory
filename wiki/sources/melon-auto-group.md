# melon-auto-group

> Truck dealership website for Melon Auto Group

## Overview

A workspace containing a Next.js 16 truck dealership website for Melon Auto Group (client: Tasos). The project is organized with the active codebase in an `auto-melon-group/` subdirectory, featuring vehicle listings powered by Supabase, component variations for layout and sections, and image processing with Sharp.

## Tech Stack

- Next.js 16 (App Router)
- TypeScript
- Supabase (database, client)
- Sharp (image processing)
- Playwright (browser automation/testing)

## Status

- **Language:** TypeScript
- **Last updated:** 2026-01-06
- **Created:** 2025-11-12
- **Visibility:** Private

## Key Features

- Truck/vehicle dealership website
- Vehicle listings with Supabase backend
- Multiple component variations (Hero, VehicleCard, Header, Footer)
- Image optimization with Sharp
- Data import scripts for vehicle inventory
- Site configuration system
