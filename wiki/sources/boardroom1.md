# boardroom1

> Premium executive strategic planning platform with real-time collaboration

## Overview

The Boardroom is an enterprise-grade strategic planning platform for executive teams. It provides collaborative digital boards for strategic frameworks including SWOT Analysis, OKR Planning, Strategic Roadmaps, and Risk Assessment Matrices, with real-time multi-user editing powered by Liveblocks and a premium black-and-gold UI.

## Tech Stack

- Next.js 15 (App Router)
- React 19
- TypeScript
- Supabase (PostgreSQL, Auth, RLS)
- Liveblocks (real-time collaboration)
- Tailwind CSS
- Radix UI (accessible primitives)
- React Hook Form + Zod (validation)
- Vercel (deployment)

## Status

- **Language:** TypeScript
- **Last updated:** 2026-01-06
- **Created:** 2025-09-25
- **Visibility:** Private

## Key Features

- SWOT Analysis, OKR, Strategic Roadmap, and Risk Assessment board templates
- Real-time collaborative editing with live cursors and selections
- Multi-tenant organization architecture with Row Level Security
- Role-based permissions (Admin, Member, Viewer)
- Command palette (Cmd+K) for power users
- Drag-and-drop board items
- Activity logging and audit trails
- Board sharing with granular permissions
- Premium black-and-gold executive UI design
