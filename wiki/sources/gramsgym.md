# gramsgym

> Bilingual gym management web app for a family-owned fitness center in Amman, Jordan

## Overview

Grams Gym is a bilingual (Arabic/English) gym management web application for a family-owned fitness center in Amman, Jordan. It provides a coach dashboard for member management, bookings, subscriptions, and schedule management; a member portal for PT booking, subscription viewing, and an AI fitness chatbot; and a public marketing website with an AI chatbot widget.

## Tech Stack

- Next.js 16.1 App Router, React 19, TypeScript
- Supabase (PostgreSQL with RLS)
- Tailwind CSS 4, shadcn/ui (new-york style)
- OpenRouter API (Mistral model for AI chat)
- Framer Motion (animations)
- Sentry, Vercel Analytics, Speed Insights
- Resend (email notifications)
- Vitest (testing)
