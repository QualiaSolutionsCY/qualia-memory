# haamah

> Haamah — client website built on Next.js 16 with Framer Motion animations and markdown content.

## Overview

A Next.js 16 website for Haamah. Built with Framer Motion for animations, Lucide icons, and markdown rendering (react-markdown + remark-gfm). The project includes an API layer and uses a modern frontend stack with Tailwind CSS and utility libraries (clsx, tailwind-merge).

## Tech Stack

- Next.js 16.1, React 19, TypeScript
- Tailwind CSS (with tailwind-merge, clsx)
- Framer Motion (animations)
- Lucide React (icons)
- react-markdown + remark-gfm (content rendering)
- Vercel (hosting)

## Status

- **Language:** TypeScript
- **Last updated:** 2026-03-25
- **Visibility:** Private

## Key Features

- Animated UI with Framer Motion
- Markdown-based content rendering
- API routes
- Modern responsive design with Tailwind CSS
