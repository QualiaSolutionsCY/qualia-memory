# dababneh

> Invoice generator for SS Law firm — create invoices with line items, auto-calc dates, client management, PDF export

## Overview

Dababneh (SS Law) is an invoice generation tool built for a law firm. It allows creating invoices with line items, automatic date calculations, client management, and PDF export. The app includes a dashboard with invoice overview and an admin panel for management.

## Tech Stack

- Next.js 15, React 19, TypeScript
- Supabase (client storage, invoices, auth)
- jsPDF (PDF generation), Recharts (charts)
- Tailwind CSS
- Vercel (deployment)

## Status

- **Language:** TypeScript
- **Last updated:** 2026-03-25
- **Visibility:** Private

## Key Features

- Invoice creation with line items and auto-calculated dates
- Client management and contact info storage
- PDF export via jsPDF
- Dashboard with invoice overview and charts (Recharts)
- Admin panel
- Supabase-backed data storage with RLS
