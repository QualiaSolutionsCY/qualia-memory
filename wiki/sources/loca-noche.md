# loca-noche

> Event ticketing platform for Oktoberfest events in Cyprus

## Overview

LocaNoche is a Next.js-based event ticketing platform specifically built for Oktoberfest events in Cyprus. It integrates with VivaPayments for payment processing and N8N for automated ticket generation and email delivery. Uses Prisma ORM for database management with full test coverage via Jest and Playwright.

## Tech Stack

- Next.js (App Router)
- TypeScript
- Prisma ORM (database)
- VivaPayments (payment processing)
- N8N (workflow automation for ticket generation/email)
- Jest (unit testing)
- Playwright (E2E testing)
- Docker Compose
- Tailwind CSS / PostCSS
