# carton

> Voice AI lead follow-up system for medical practices — AI voice agent that calls leads and books appointments

## Overview

Carton is a Voice AI Lead Follow-up System built for Concept Carton, a UK marketing agency. It solves the problem of medical practices not following up with 40-50% of leads by deploying an AI voice agent named Sophie that calls leads and books appointments. The system integrates with VAPI for voice AI, ElevenLabs for TTS, Deepgram for STT, and GoHighLevel as the CRM source.

## Tech Stack

- Next.js 15, React 19, TypeScript, Tailwind CSS
- shadcn/ui
- Supabase (PostgreSQL, Edge Functions)
- VAPI (voice AI orchestration)
- ElevenLabs (TTS), Deepgram Nova-3 (STT)
- GoHighLevel (CRM source)
- Vercel (deployment)
