# giannino-mayfair

> Giannino Mayfair — Italian restaurant since 1899 — Next.js 16 website rebuild.

## Overview

A modern rebuild of the Giannino Mayfair marketing website. Giannino is a luxury Italian restaurant at 10 Blenheim St, Mayfair, London, with heritage dating back to Milan in 1899. The project replaces the existing WordPress/Elementor site with a Next.js 16 build, preserving the heritage narrative while modernizing layout, typography, and motion to match the restaurant's luxury positioning.

## Tech Stack

- Next.js 16, React 19, TypeScript
- Tailwind CSS
- OpenTable integration (reservations)
- CMS for client-editable content (hours, menus, events)
- Vercel (hosting)

## Status

- **Language:** TypeScript
- **Last updated:** 2026-04-14
- **Visibility:** Private

## Key Features

- Heritage-driven editorial design (1899 Milan to London story)
- OpenTable reservation integration throughout
- Private dining enquiry system (4 spaces: The Snug, Lobby Bar, Dining Room, Private Dining Room)
- Structured menu pages with CMS editing capability
- About/Heritage page with historic imagery
- Contact strip with phone, WhatsApp, email, social links
- Responsive, cinematic layout replacing dated WordPress site
