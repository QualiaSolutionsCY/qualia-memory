# theracoach

> American English speech practice assistant — gamified, evidence-based speech exercises for children and youth

## Overview

Thera Coach is an American English speech practice assistant providing gamified, evidence-based speech exercises for children (ages 5-10) and youth (ages 11-17). It covers fluency, articulation, and functional communication goals through an AI chat interface powered by OpenRouter with Gemini 2.5 Pro Preview, using streaming responses.

## Tech Stack

- Next.js 16 App Router, React 19, TypeScript
- Tailwind CSS
- OpenRouter API (Gemini 2.5 Pro Preview model)
- Streaming responses
- Jest (testing)

## Status

- **Language:** TypeScript
- **Last updated:** 2026-03-03
- **Visibility:** Private

## Key Features

- Gamified speech exercises for two age groups (5-10, 11-17)
- Evidence-based fluency, articulation, and functional communication practice
- AI chat interface with streaming responses
- System prompt with persona overlays for age-appropriate interaction
- OpenRouter API integration with Gemini 2.5 Pro Preview
