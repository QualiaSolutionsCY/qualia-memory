# Vuet

> Flutter mobile app with Supabase backend and Firebase integration

## Overview

Vuet is a cross-platform mobile application built with Flutter/Dart. It uses Supabase for backend services and authentication, with Firebase Core integration. The app targets Android, iOS, web, macOS, and Linux platforms.

## Tech Stack

- Flutter (Dart SDK >=3.2.3)
- Supabase Flutter SDK (backend, auth)
- Firebase Core
- Google Fonts
- Provider (state management)
- Flutter Secure Storage / Shared Preferences
- Flutter Local Notifications
- Codemagic (CI/CD)
