# pastrikos-solar-demo

> Pastrikos Solar Parts Platform — B2B e-commerce demo for solar panel installers with installation calculator.

## Overview

A demo B2B e-commerce platform for A. Pastrikos LTD (Paphos, Cyprus). The platform allows solar panel installers to browse parts, get account-based tiered pricing, and use an installation calculator that computes exact parts needed based on panel count, panel type, and roof type. Includes both an installer-facing storefront and an admin panel for Pastrikos staff.

## Tech Stack

- Next.js, React 19, TypeScript
- Tailwind CSS
- Vercel (hosting)

## Status

- **Language:** TypeScript
- **Last updated:** 2026-03-26
- **Visibility:** Private

## Key Features

- Searchable parts catalog with categories
- Account-based login for installers
- Tiered pricing per customer relationship
- One-time discount codes
- Installation calculator: panels x type x roof = parts list + quantities
- Add calculated parts directly to cart
- Order placement (pickup or shipping)
- Admin panel: manage products, pricing tiers, discount codes
- Admin panel: manage customer accounts and discount tiers
- Responsive PWA (installable from browser)
