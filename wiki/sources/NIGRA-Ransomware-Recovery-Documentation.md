# NIGRA-Ransomware-Recovery-Documentation

> Incident response documentation for NIGRA ransomware recovery (July 2025)

## Overview

Documentation repository for a business-critical ransomware incident that occurred on July 25, 2025. Contains a structured 6-phase recovery plan, technical forensic analysis, business impact assessment, and investigation checklists for recovering a PrismaNET ERP system with encrypted SQL Server databases and business files (.nigra extension).

## Tech Stack

- HTML/Markdown (documentation)
- SQL Server (affected database)
- PrismaNET ERP (affected business system)
- Emsisoft Decryption Tools
- Key Metric SQL Backup Master
- Windows Volume Shadow Copy
